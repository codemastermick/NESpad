NES Controller Library for Arduino

A simplified interface for an original NES controller, that provides a small file size yet a large number of features.

Inspired by https://github.com/dfanara/NESDuino/tree/master/NESDuino
Redesigned by Mick Evans <codemastermick@gmail.com>  November 20th 2019

See CHANGELOG.txt for version information

Button Mapping:
0 = a
1 = b
2 = select
3 = start
4 = up
5 = down
6 = left
7 = right
